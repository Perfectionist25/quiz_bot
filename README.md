# QuizzBot

Telegram-бот для тестов из `.txt` файлов:
- загрузка тестов пользователями;
- список всех тестов и список своих тестов;
- индивидуальное прохождение (каждый пользователь проходит отдельно);
- выбор таймера на каждый вопрос (`без таймера`, `10/15/20/30/45/60` секунд);
- итоговая подробная статистика по прохождению;
- личная статистика и общая статистика по всем тестам.

## Запуск

1. Создайте бота через `@BotFather` и получите токен.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Установите переменную окружения или создайте файл `.env`:

```bash
export BOT_TOKEN="ваш_токен"
```

или

```bash
echo 'BOT_TOKEN="ваш_токен"' > .env
```

4. Запустите:

```bash
python bot.py
```

После первого запуска рядом появится база `quizzbot.db`.

## Формат TXT

Поддерживается гибкий формат. Пример:

```txt
# Название теста (необязательно)
1) Первый вопрос
- Неправильный вариант
+ Правильный вариант
- Еще один вариант

2) Второй вопрос
- Ответ A
- Ответ B *
- Ответ C
```

Правильные ответы можно пометить любым из трёх способов:
- `+` в начале строки;
- `*` в конце строки;
- `(+)` в конце строки.

Требования:
- файл в UTF-8;
- минимум 2 варианта ответа на каждый вопрос;
- минимум 1 правильный вариант на вопрос;
- размер до 1MB.

Файл должен быть текстовым `.txt`. Порядок вариантов не важен — бот показывает ответы в случайном порядке, поэтому правильный ответ не будет всегда первым.

## Кнопки в боте

- `📚 Список тестов`
- `➕ Создать тест из TXT`
- `🗂 Мои тесты`
- `📊 Моя статистика`
- `🌐 Общая статистика`
- `ℹ️ Формат TXT`

Во время теста:
- кнопки с вариантами ответа;
- `⛔ Завершить тест`.

После теста:
- `🔁 Пройти снова`
- `📚 К списку тестов`
- `🏠 Главное меню`
